# Web의 기본 3가지 요소

## URI(Uniform Resource Identifier)
* 리소스 식별자
* 모든 정보에 접근할 수 있는 정보


## HTTP(Hypertext Transfer Protocol)
어떠한 방식으로 해당 데이터를 주고받을지에 대한 규약
1. GET
2. POST
3. PUT
4. DELETE
5. OPTIONS
6. HEAD
7. TRACE
8. CONNECT
